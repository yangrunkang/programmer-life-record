### 四、相关问题QA

#### 1. 为什么起名为 "程序员的生活记录" ?

**程序员的生活记录** 是我刚开始工作时为了记录我的生活起的一个名字,没想到一直用到了现在。围绕**程序员的生活记录**我发布了很多电台,所以在2022年开年想以**程序员的生活记录**为主题创作更多的内容!

#### 2. 内容主要受众有哪些呢?

- 即将要找工作的人

- 喜欢程序的爱好者,特别是Java语言爱好者

- 喜欢设计系统架构的人,喜欢折腾的人,哈哈

- 应届生,因为内容难度都不是很大(哈哈,主要是也写不了太难的内容),很容易理解和接手
